Weather App

Project Overview

a responsive weather app that displays real-time temperature for any location worldwide. Integrated light and dark mode features for user preference, along with the ability to toggle between Celsius and Fahrenheit. The app offers a clean and interactive user interface, enhancing the user experience.

Key Features

  • Responsive Design
  • User Engagement
  • Menu

Frontend: HTML, CSS, JavaScript, Weather API

Future Enhancements

Recipe Video Tutorials: Add video tutorials for popular recipes to provide visual guidance.

Custom Spice Blends: Introduce features for creating and sharing custom spice blends.

International Recipes: Expand the recipe database to include spicy dishes from various international cuisines.
